# Usage
```yaml
user:
  dom0:
    - packages.split-ssh.dom0
  
  template-development:
    - packages.split-ssh.client-template
  
  template-vault:
    - packages.split-ssh.vault
  
  development:
    - packages.split-ssh.client-appvm
```

Follow this guide to add keys to KeePassXC:
